How to run verification

  1. Open the Universe Hub in a new tab: index.html
  2. Hard refresh the hub tab with Ctrl+Shift+R (or append ?cb=timestamp to the URL)
  3. Open the browser console (F12 or Cmd+Option+J on macOS)
  4. Copy and paste the verification script below into the console, then press Enter
Tip

Keep this dashboard open in its own tab. All checks run inside the hub context via the console script to avoid cross-context false negatives.

Visitor Tracker API

Expected console output after running the script in the hub tab:

  • Visitor tracker exists with getVisitedWorlds() returning an array
  • getVisitorId() returns a string (printed as first 8 chars)
  • No missing exports or undefined errors
Manual follow-up

Use the Directory Visit buttons to confirm teleport still works without tracker errors.

Event Witness Challenge

Expected console output:

  • Challenge UI present with eventWitness progress displayed
  • Instruction to trigger window.UniverseEvents.triggerEvent('aurora') manually
  • No missing UniverseEvents API errors
Manual follow-up

Run window.UniverseEvents.triggerEvent('aurora') in the hub console to confirm progress increments.
